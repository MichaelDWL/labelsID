Como usar

  1. Escolha o modelo de etiqueta — Pequena (4,5×1,5 cm), Média (6,5×2 cm) ou Grande/Placa (18×7 cm).
  2. Envie as logos (opcional) — Até 2 imagens em PNG por etiqueta. Faça isso antes de escolher o arquivo Excel; depois de enviar a planilha, as logos ficam bloqueadas.
  3. Selecione o arquivo Excel — Use o botão "Escolher arquivo Excel" e escolha um arquivo .xlsx ou .xls. Cada linha da planilha vira uma etiqueta.
  4. Gere o PDF — Clique em "Gerar PDF das etiquetas". O arquivo será baixado com as etiquetas prontas para impressão em folha A4.

Modelos de etiqueta

Escolha o tamanho conforme o uso: bins menores, médios ou placas/avisos.

Etiqueta pequena

Dimensão: 4,5 × 1,5 cm

Ideal para bins menores (ex.: nº3). Várias etiquetas por folha A4.

Etiqueta média

Dimensão: 6,5 × 2 cm

Para bins a partir do nº5 e identificações médias.

Etiqueta grande (placa)

Dimensão: 18 × 7 cm

Para placas, avisos e áreas que precisam de maior destaque.

Preparando a planilha

  • Formato: Excel (.xlsx ou .xls). Use a primeira aba da planilha.
  • Uma linha = uma etiqueta. A primeira linha pode ser o cabeçalho; se quiser que ela vire etiqueta, deixe como as demais.
  • Várias colunas: o conteúdo das células da mesma linha é unido com " - " (espaço, traço, espaço). Ex.: coluna A "Setor" e coluna B "Prateleira 2" geram a etiqueta: Setor - Prateleira 2.
  • Células vazias são ignoradas. Linhas totalmente vazias não geram etiqueta.

Exemplo de planilha

Nome
Material A
Material B

Resultado: etiqueta 1 = "Material A "; etiqueta 2 = "Material B".

Logos (imagens)

  • Formato: PNG (recomendado com fundo transparente).
  • Quantidade: até 2 logos por etiqueta.
  • Ordem: envie as logos antes de selecionar o arquivo Excel. Depois que a planilha for escolhida, o envio de logos fica bloqueado para aquela seção.

Tamanhos recomendados para boa nitidez na impressão:

Modelo de etiqueta Tamanho recomendado (pixels)
Pequena (4,5×1,5 cm) 60 × 60 px
Média (6,5×2 cm) 80 × 80 px
Grande / Placa (18×7 cm) 150 × 80 px

Imagens maiores podem ser redimensionadas pelo navegador. Use PNG com fundo transparente para melhor resultado.

Dicas

  • Prepare as logos em PNG e envie primeiro; só depois selecione o Excel.
  • Na planilha, evite textos muito longos em uma única célula para caber bem na etiqueta.
  • O PDF é gerado em tamanho real para impressão em folha A4; confira a pré-visualização antes de imprimir.
