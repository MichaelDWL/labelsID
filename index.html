Gerador de Etiquetas Automático

Crie etiquetas padronizadas em segundos a partir de uma planilha de Excel ou XML.

Selecione o modelo da Etiqueta

Etiqueta Pequena

4,5 x 1,5 cm

Ideal para bins menores, como os de nº3, este modelo oferece uma solução compacta e prática para identificação.

Dimensão da etiqueta: 4,5 × 1,5 cm.

Etiqueta Média

6,5 x 2 cm

Pensado para bins a partir do nº5, este modelo atende perfeitamente etiquetas de tamanho médio e também funciona muito bem em identificações maiores.

Dimensão da etiqueta: 6,5 × 2 cm.

Etiqueta Grande

18 x 7 cm

Desenvolvido para identificações de áreas, placas e avisos, este modelo é ideal quando é necessário maior destaque e visibilidade.

Dimensão da etiqueta: 18 × 7 cm.
